# Walter the Walker (Fit Tracker)

App Android per monitorare camminate, circuito pesi e statistiche per persone senior.

## Moduli principali
- Camminata con GPS e contapassi
- Circuito pesi con registrazione serie/ripetizioni/peso
- Piani di allenamento automatici e personalizzati
- Statistiche e grafici di miglioramento

## Come avviare

### Opzione 1: Android Studio
Apri il progetto in Android Studio, sincronizza Gradle e avvia su un dispositivo Android.

### Opzione 2: Riga di comando
```bash
# Compilare l'app
./gradlew assembleDebug

# Installare su dispositivo connesso
./gradlew installDebug
```

## Requisiti
- Android SDK (API 26+)
- JDK 17 o superiore
- Dispositivo Android con Android 8.0 (Oreo) o superiore
